# DevTrack AI

DevTrack AI is a Developer Productivity Intelligence Dashboard designed to help developers track projects, log sessions, and analyze productivity patterns.

## Current Features (v0.1)

- Responsive dashboard layout
- Project creation and storage (LocalStorage)
- Dynamic dashboard metrics
- Dark SaaS-style UI

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage (Simulated Persistence)
